# Predição de Churn de Clientes

## Objetivo

Desenvolver um modelo preditivo capaz de identificar clientes com maior probabilidade de cancelamento (churn).

## Tecnologias

- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

Telco Customer Churn Dataset

## Etapas

- Limpeza dos dados
- Análise exploratória
- Tratamento de variáveis
- Feature Engineering
- Treinamento de modelo preditivo
- Avaliação das métricas

## Resultados

- Accuracy: 81%
- Recall para churn: 58%

## Principais Insights

### Reduzem churn

- Maior tempo de permanência
- Contratos anuais
- Contratos de dois anos
- Suporte técnico

### Aumentam churn

- Mensalidades elevadas
- Serviço de fibra óptica
- Pagamento via Electronic Check
